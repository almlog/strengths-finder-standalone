## 技術的知見

### git rm --cached の活用

```bash
# リポジトリからのみ削除（ローカルファイルは保持）
git rm --cached <file>

# ディレクトリの場合は -r オプション
git rm -r --cached <directory>
```

**メリット:**
- ローカルの開発環境に影響を与えずにリポジトリをクリーンアップ
- .gitignoreと組み合わせて今後の追跡を防止
