## システム概要

チームメンバーの**強み（ストレングスファインダー34資質）**と**性格（16Personalities/MBTI）**を統合分析し、チームビルディングや人材配置を支援するWebアプリケーションです。

### コンセプト
- **StrengthsFinder**（何が得意か）を主軸に
- **MBTI**（どう考えるか）を補助軸として
- 多角的なメンバー分析を提供

### プライバシー重視
- 完全クライアントサイド動作
- 個人情報は外部サーバーに送信されない
- データはブラウザ内に保存

## 分析スコア一覧

### プロファイル分析（個人）

| スコア | 説明 | 計算基盤 |
|--------|------|----------|
| 相性スコア | MBTI × Top5資質の適合度（0-100） | 独自マトリクス |
| チーム適合度 | チーム協働への適性（0-100） | Belbin 9ロール理論 |
| リーダーシップ潜在力 | リーダーシップ資質（0-100） | Bass変革型リーダーシップ理論 |
| 役割タイプ | 16パターンの自動推論 | MBTI×資質カテゴリ |

### チーム分析（部署・グループ）

| 指標 | 説明 |
|------|------|
| 強みグループ分布 | 実行力・影響力・人間関係構築力・戦略的思考力の4カテゴリ割合 |
| 頻出資質TOP10 | チーム内で多く保有される資質ランキング |
| チームタイプ判定 | バランス型・特化型・複合型・中心型の自動分類 |
| チーム特性ナラティブ | 資質分析に基づく自動生成テキスト |

## クイックリファレンス

### StrengthsFinder 34資質（4カテゴリ）

| カテゴリ | 資質 |
|---------|------|
| **実行力** | 達成欲、アレンジ、信念、公平性、慎重さ、規律性、目標志向、責任感、回復志向 |
| **影響力** | 活発性、指令性、コミュニケーション、競争性、最上志向、自己確信、自我、社交性 |
| **人間関係構築力** | 適応性、運命思考、成長促進、共感性、調和性、包含、個別化、ポジティブ、親密性 |
| **戦略的思考力** | 分析思考、原点思考、未来志向、着想、収集心、内省、学習欲、戦略性 |

### 16Personalities（MBTI）16タイプ

| グループ | タイプ |
|---------|-------|
| **分析家** | INTJ, INTP, ENTJ, ENTP |
