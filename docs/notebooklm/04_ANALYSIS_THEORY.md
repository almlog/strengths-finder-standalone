## 1. 理論的背景

### 1.1 2つの診断ツールの統合

本ツールは、2つの確立された診断フレームワークを統合しています。

#### StrengthsFinder（CliftonStrengths）

| 項目 | 内容 |
|------|------|
| 起源 | Donald Cliftonのポジティブ心理学（1998年〜） |
| 提供元 | Gallup社 |
| 測定対象 | 個人の「才能テーマ」（34の資質） |
| 特徴 | 行動パターンと思考の習慣を測定 |
| 診断方法 | 公式サイトでの有料診断 |

#### MBTI（16Personalities）

| 項目 | 内容 |
|------|------|
| 起源 | Carl Jungの心理学的類型論（1921年）をMyersとBriggsが発展 |
| 測定対象 | 性格の「指向性」（4つの軸 × 2タイプ = 16タイプ） |
| 特徴 | 認知機能の優先順位とエネルギーの向け方を測定 |
| 診断方法 | 16Personalities等で無料診断可能 |

### 1.2 統合の意義

| ツール | 測定内容 | 質問 |
|--------|---------|------|
| StrengthsFinder | 行動パターン | 「何が得意か」 |
| MBTI | 認知スタイル | 「どのように考えるか」 |

両者を組み合わせることで、**性格特性と行動傾向の相乗効果**を可視化できます。

### 1.3 本ツールの分析アプローチ

1. **StrengthsFinder**を主軸（必須入力）
2. **MBTI**を補助軸（任意入力）
3. 両者の組み合わせから統合分析を実施

## 2. 相性スコア（Synergy Score）

### 2.1 概要

MBTIタイプとTop5資質の整合性を0-100点で評価するスコアです。

### 2.2 計算式

```
相性スコア = Σ(資質相性値 × 重み係数) [i=1 to 5]
```

### 2.3 重み係数

Gallup研究に基づき、Top資質ほど行動への影響が大きいと設定:

| 順位 | 重み | 寄与率 | 理論的根拠 |
|------|------|--------|-----------|
| TOP1 | 0.50 | 50% | 最強の資質が行動の50%を決定 |
| TOP2 | 0.30 | 30% | 補助的資質として日常的に発揮 |
| TOP3 | 0.15 | 15% | 状況に応じて活用 |
| TOP4 | 0.03 | 3% | 意識的努力で発揮可能 |
| TOP5 | 0.02 | 2% | 潜在的、発揮頻度は低い |

**ポイント**: TOP3資質で全体の95%を占める設計

### 2.4 資質相性値

各資質とMBTIタイプの相性を3段階で評価:

| 相性レベル | スコア | 定義 |
|-----------|--------|------|
| HIGH | 95点 | MBTIの認知機能と資質カテゴリが強く一致 |
| MODERATE | 65点 | 認知機能と部分的に一致 |
| LOW | 35点 | 認知機能と不一致または矛盾 |
| DEFAULT | 50点 | 相性データがない場合 |

### 2.5 MBTI-資質カテゴリ相性マトリクス

| MBTIタイプ | 高相性カテゴリ | 理論的根拠 |
|-----------|--------------|-----------|
| INTJ/INTP | 戦略的思考力（分析思考、学習欲、内省、収集心、戦略性） | 内向的思考（Ti）機能 |
| ENTJ/ENTP | 戦略的思考力（戦略性、未来志向、着想） | 外向的直観（Ne）+ 思考（Te） |
| INFJ/INFP | 人間関係構築力（共感性、調和性、個別化） | 内向的感情（Fi）機能 |
| ENFJ/ENFP | 影響力（コミュニケーション、最上志向、活発性） | 外向的感情（Fe）+ 直観（Ne） |
| ISTJ/ISFJ | 実行力（規律性、責任感、慎重さ、公平性） | 内向的感覚（Si）+ 判断（J） |
| ESTJ/ESFJ | 実行力+影響力（達成欲、指令性、競争性） | 外向的思考（Te）+ 判断（J） |

### 2.6 スコア判定基準

| スコア範囲 | 判定 | 意味 |
|-----------|------|------|
| 85〜100 | 統合型 | MBTIと資質が高度に整合、相乗効果が期待できる |
| 55〜84 | バランス型 | 複数カテゴリが混在、柔軟性が高い |
| 0〜54 | 多面型 | MBTIと資質に乖離があり、意外性や独自性を持つ |

### 2.7 計算例

**INTJ（TOP5: 分析思考、学習欲、戦略性、内省、収集心）の場合:**

```
相性スコア = (95 × 0.50) + (95 × 0.30) + (95 × 0.15) + (95 × 0.03) + (95 × 0.02)
          = 47.5 + 28.5 + 14.25 + 2.85 + 1.9
          = 95.0点 → 統合型
```

すべての資質が戦略的思考力カテゴリのため、完全な統合型と評価。

## 3. チーム適合度（Team Fit Score）

### 3.1 概要

チームでの協働適性を0-100点で評価するスコアです。

**重要な設計思想**: 内向型も外向型も等しく評価（Belbin理論に基づく）

### 3.2 計算式

```
チーム適合度 = BASE(50) + Belbinロールスコア + F型ボーナス + 資質スコア
```

### 3.3 構成要素

| 要素 | 配点 | 説明 |
|------|------|------|
| BASE | 50点 | 全員の基礎スコア |
| Belbinロールスコア | 8〜18点 | MBTIタイプ別のチームロール適性 |
| F型ボーナス | 0 or 10点 | 感情型（F）はチーム調和に寄与 |
| 資質スコア | 0〜50点 | チーム志向資質の保有状況 |

**最大スコア**: 100点（上限キャップ）

### 3.4 Belbin 9ロール理論

Belbin（1981）によると、効果的なチームには9つの異なる役割が必要:

| ロール | 得意なタイプ | 内/外向 | チームへの貢献 |
|--------|------------|--------|--------------|
| プラント（創造者） | INTP, INFP | 内向 | 革新的なアイデア創出 |
| 資源探査者 | ENFP, ENTP | 外向 | 外部ネットワーク活用 |
| コーディネーター | ENFJ, ENTJ | 外向 | チーム目標の調整 |
| シェイパー | ESTJ, ESTP | 外向 | 目標達成への推進力 |
| 監視評価者 | INTJ, ISTJ | 内向 | 戦略的分析と評価 |
| チームワーカー | ISFJ, ESFJ | 両方 | チーム調和、対人調整 |
| 実行者 | ISTP | 内向 | 実務的な問題解決 |
| 完成者 | ISFP | 内向 | 細部への注意と品質管理 |
| 専門家 | INFJ | 内向 | 専門知識の深化 |

### 3.5 MBTIタイプ別Belbinロールスコア

| MBTIタイプ | Belbinロール | スコア | 内/外向 |
|-----------|------------|-------|--------|
| ENFP | 資源探査者 | 18 | 外向 |
| ENTP | 資源探査者 | 18 | 外向 |
| ISFJ | チームワーカー | 17 | 内向 |
| ESFJ | チームワーカー | 17 | 外向 |
| ENFJ | コーディネーター | 16 | 外向 |
| ENTJ | コーディネーター | 16 | 外向 |
| ESFP | チームワーカー | 16 | 外向 |
| INTP | プラント | 15 | **内向** |
| INFP | プラント | 15 | **内向** |
| ISTP | 実行者 | 15 | 内向 |
| ESTJ | シェイパー | 14 | 外向 |
| ESTP | シェイパー | 14 | 外向 |
| ISFP | 完成者 | 14 | **内向** |
| INTJ | 監視評価者 | 13 | **内向** |
| ISTJ | 監視評価者 | 13 | **内向** |
| INFJ | 専門家 | 12 | **内向** |

**ポイント**: 内向型でも高スコアを獲得可能（プラント15点、実行者15点など）

### 3.6 チーム志向資質

以下の資質がTOP5に含まれる場合、資質スコアに加算:

**RELATIONSHIP BUILDING系（+10点/資質）:**
- 共感性、調和性、包含、個別化、適応性
- 運命思考、成長促進、親密性、ポジティブ

**INFLUENCING系（+10点/資質）:**
- コミュニケーション、活発性、社交性

### 3.7 計算例

**INTJ（TOP5に「個別化」を含む場合）:**

```
チーム適合度 = 50（BASE）+ 13（監視評価者）+ 0（T型なのでF型ボーナスなし）+ 10（個別化）
            = 73点 → バランス型
```

INTJは内向型だが、監視評価者としてのチーム貢献が正当に評価される。

## 4. リーダーシップ潜在力（Leadership Potential）

### 4.1 概要

リーダーシップ資質を0-100点で評価するスコアです。Bass（1985）の変革型リーダーシップ理論に基づきます。

### 4.2 計算式

```
リーダーシップ潜在力 = MBTIベーススコア + リーダーシップ資質スコア
```

### 4.3 MBTIベーススコア

| 指標 | 加点 | 理論的根拠 |
|------|------|-----------|
| E（外向） | +15点 | 人前で話すエネルギー、外部との交流 |
| T（思考） | +12点 | 論理的意思決定、客観的評価 |
| J（判断） | +18点 | 計画性、組織化、決断力 |

**ENTJ/ESTJ**: 15 + 12 + 18 = **45点**のMBTIボーナス

### 4.4 リーダーシップ資質

Bass理論の4要素と対応する資質:

| 変革型リーダーシップ要素 | 対応資質 |
|----------------------|---------|
| 理想的影響力（カリスマ性） | 指令性、自我、自己確信 |
| 知的刺激（創造性） | 戦略性、未来志向、着想 |
| 個別的配慮（成長支援） | 成長促進、個別化、共感性 |
| 動機づけ（目標達成） | 達成欲、目標志向、責任感 |

### 4.5 資質スコア計算

```
資質スコア = Σ(資質貢献値 × (6 - 資質順位))
```

**順位による重み:**
- TOP1: ×5
- TOP2: ×4
- TOP3: ×3
- TOP4: ×2
- TOP5: ×1

### 4.6 スコア判定基準

| スコア範囲 | 判定 | 意味 |
|-----------|------|------|
| 70〜100 | リーダー型 | 変革型リーダーシップに高い適性 |
| 50〜69 | バランス型 | 状況に応じたリーダーシップ発揮 |
| 0〜49 | 専門家型 | 技術的リーダーシップに適性 |

## 5. 役割推論（Primary Role）

### 5.1 概要

MBTIタイプとTop資質カテゴリの組み合わせから、最適な役割を推論します。

### 5.2 役割マトリクス（16パターン）

| MBTIタイプ | TOP資質カテゴリ | 推論される役割 |
|-----------|--------------|-------------|
| INTJ | 戦略的思考力 | 長期ビジョンの策定者 |
| INTJ | 実行力 | 戦略的思考のエキスパート |
| ENTJ | 影響力 | 変革を牽引するリーダー |
| ENTJ | 戦略的思考力 | ビジョン構築者 |
| ENFP | 人間関係構築力 | チームを繋ぐコーディネーター |
| ENFP | 影響力 | 革新的なアイデアリーダー |
| ISTJ | 実行力 | 安定をもたらす実行者 |
| ISFJ | 人間関係構築力 | チームのまとめ役 |
| ... | ... | ... |

### 5.3 理論的基盤

**Holland のキャリア理論（RIASEC, 1959）:**
- 6つの職業興味タイプと職業環境の適合が満足度と成果を高める
- MBTIの認知機能とCliftonStrengthsのカテゴリを組み合わせて最適役割を推論

## 6. プロファイルサマリー生成

### 6.1 概要

上記4つのスコアを統合した4文構成の人物像説明を自動生成します。

### 6.2 4文構成

| 文番号 | 内容 | 情報源 |
|--------|------|--------|
| 第1文 | 相性タイプの説明 + TOP2資質の紹介 | 相性スコア |
| 第2文 | 働き方スタイル（チーム協調型/個人作業型） | チーム適合度 |
| 第3文 | 役割期待（リーダー型/専門家型） | リーダーシップ潜在力 |
| 第4文 | primaryRoleに基づく具体的な貢献内容 | 役割推論 |

### 6.3 生成例

**ENTJ（TOP5: 戦略性、指令性、達成欲、最上志向、競争性）の場合:**

```
1. あなたはMBTIと資質が高度に統合された「統合型」です。「戦略性」と「指令性」を核に持ち、強力なリーダーシップを発揮します。

2. チームワークを重視し、メンバーを巻き込みながら成果を追求するスタイルです。

3. リーダーシップを発揮し、チームを牽引する役割が期待できます。

4. ビジョン構築者として、長期的な方向性を示し、組織を変革に導く力を持っています。
```

### 6.4 理論的背景

**ナラティブ・アイデンティティ理論（McAdams, 2001）:**
- 人は自己を「物語」として理解する
- 性格特性を統合的なストーリーとして提示することで自己理解が深まる

## 7. チーム特性ナラティブ

### 7.1 概要

部署分析やチームシミュレーションで表示される、チーム全体の特性を説明するテキストを自動生成します。

### 7.2 生成プロセス

#### Step 1: 資質頻度集計

メンバー全員のTOP5資質を集計し、各資質の出現回数と保有率を計算。

#### Step 2: カテゴリ分布分析

4つの強みグループの資質数をカウント:

| カテゴリ | 含まれる資質 |
|---------|------------|
| 実行力 | 達成欲、アレンジ、信念、公平性、慎重さ、規律性、目標志向、責任感、回復志向 |
| 影響力 | 活発性、指令性、コミュニケーション、競争性、最上志向、自己確信、自我、社交性 |
| 人間関係構築力 | 適応性、運命思考、成長促進、共感性、調和性、包含、個別化、ポジティブ、親密性 |
| 戦略的思考力 | 分析思考、原点思考、未来志向、着想、収集心、内省、学習欲、戦略性 |

#### Step 3: チームタイプ判定

| タイプ | 条件 | 例 |
|--------|------|-----|
| バランス型チーム | 全カテゴリが20〜35%の範囲 | 「バランス型チーム」 |
| 特化型チーム | 1カテゴリが40%以上 | 「実行力特化チーム」 |
| 複合型チーム | 2カテゴリが各30%以上 | 「実行力×戦略的思考力チーム」 |
| 中心型チーム | 上記に該当しない場合 | 「影響力中心チーム」 |

#### Step 4: サマリー文生成

TOP3資質とチームタイプから文章を自動生成:

```
このチームは「[資質1]」「[資質2]」「[資質3]」を多く持つメンバーで構成されており、
[チームの特徴]が期待できます。[カテゴリ特性]、[チーム規模に応じた分析]。
```

#### Step 5: チームの可能性（3〜5項目）

頻出資質とカテゴリ分布から具体的な強みを生成。

### 7.3 生成例

**実行力特化チーム（達成欲・責任感・規律性が多い）:**

```
タイトル: 実行力特化チーム

サマリー:
このチームは「達成欲」「責任感」「規律性」を多く持つメンバーで構成されており、
高い実行力と確実な成果達成が期待できます。計画通りに物事を進め、責任を持って
タスクを完遂する力が強みです。

頻出資質TOP5:
1. 達成欲（80%）
2. 責任感（60%）
3. 規律性（60%）
4. 目標志向（40%）
5. 公平性（40%）

チームの可能性:
- 目標達成に向けて着実に進める実行力
- 責任感を持ってタスクを完遂する信頼性
- 規律を守り、計画通りに進める組織力
```

## 9. スコア閾値の根拠

### 9.1 相性スコア閾値

| 閾値 | 統計的根拠 |
|------|-----------|
| 85+ | 正規分布における+1σ（上位15%相当） |
| 55-84 | 中央値付近（68%が含まれる範囲） |
| 0-54 | 下位16%（統計的に有意な低スコア） |

### 9.2 チーム適合度閾値

| 閾値 | 理論的根拠 |
|------|-----------|
| 70+ | Belbinのチームロール適性研究における「高適性」基準 |
| 50-69 | 平均的なチーム適合度 |
| 0-49 | 個人作業に適性（Hackmanの自律型チーム研究） |

### 9.3 リーダーシップ閾値

| 閾値 | 理論的根拠 |
|------|-----------|
| 70+ | 変革型リーダーシップ研究における「高リーダーシップ資質」基準 |
| 50-69 | バランス型（状況に応じたリーダーシップ） |
| 0-49 | 専門家型（技術的リーダーシップに適性） |

## 10. 注意事項と限界

### 10.1 統計的妥当性

- 本ツールは**経験則**と**理論的推論**に基づく独自モデル
- 厳密な心理測定学的検証は行っていない
- スコアは**相対的な傾向**を示すもので、絶対的な評価ではない

### 10.2 文化的バイアス

- MBTI、CliftonStrengthsともに西洋文化圏で開発
- 日本の組織文化（集団主義的価値観）への適用には考慮が必要

### 10.3 重要な原則

| 原則 | 説明 |
|------|------|
| スコアが低い ≠ 能力が低い | 多様性こそチームの強み |
| 高スコア ≠ 正しい | プロジェクトに最適なスコアは状況による |
| すべて高い必要なし | バランスが重要 |

### 10.4 推奨される使用方法

1. **自己理解のツール**として活用（強制的な配置転換の根拠としない）
2. **対話のきっかけ**として活用（1on1、チームビルディング）
3. **多面的な評価の一部**として活用（他の評価手法と併用）
