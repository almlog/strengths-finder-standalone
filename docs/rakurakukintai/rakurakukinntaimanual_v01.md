# 楽楽勤怠データ仕様・運用ガイド（WEBシステム掲載用）

## 1. データ集計と締め日の仕様
勤怠データの確定タイミングと集計の基準です。分析システム上のデータ鮮度に関わります。

*   **月次データの確定:**
    *   **締め切り:** 翌月第1営業日の13:00まで,。
    *   **集計基準:** 13:00時点のデータを基に計算が行われます。
    *   **概算集計:** 月末2営業日前に概算集計が行われるため、前日までの入力が必要です。
*   **データのロック:**
    *   締め処理後はデータがロックされます。締め後の修正は、上長承認および総務によるロック解除が必要です。

## 2. 画面表示・ステータスの定義
楽楽勤怠からエクスポートされたデータや画面上の表示ステータスの意味です。

### 申請ステータス（背景色・承認印）
| ステータス | 画面上の表示色 | 説明 |
| :--- | :--- | :--- |
| **申請中** | 赤色 (背景) | 申請提出済み、承認ルートで決裁待ちの状態, |
| **承認済** | 黒色 (承認印) / 灰色 (背景) | 最終承認者が承認を完了した状態, |
| **システム登録** | 白色 (背景) | システム管理者が直接登録したデータ |
| **修正・取消中** | 灰色 (背景) | 承認済みの申請に対して、修正または取消申請中の状態 |

### アラート（入力不備）
*   **表示:** 出勤簿上の「赤い三角マーク」やトップページのアラート一覧に表示されます,。
*   **影響:** アラートが残っている状態では「出勤簿提出（月締め）」ができません,。
*   **対応:** 対象日をクリックし、表示されるメッセージに従って「打刻修正申請」等を行う必要があります,。

## 3. 勤務区分別 計算・入力ロジック
分析システムで「残業時間」や「休憩時間」を算出・可視化する際の前提となる入力ルールです。

### A. 残業・早出の判定ロジック
勤務形態（客先常駐か内勤か）によって、残業計算のトリガーが異なります。

| 勤務形態 | 早出残業の計算方法 | 定時後残業の計算方法 |
| :--- | :--- | :--- |
| **客先就業者** | 出勤簿の「早出フラグ」欄に「1」を入力すると計算対象となる, | 退社時刻から自動計算される |
| **内勤者** | 「早出申請」の提出が必須（フラグ入力では計算されない） | 「残業申請」の提出が必須 |

### B. 休憩時間の計算ルール
*   **法定基準:** 労働時間が6時間を超える場合は45分以上、8時間を超える場合は60分以上の休憩が必要です,。
*   **自動計算と修正:** 基本設定（例：12:00-13:00）以外で休憩を取った場合（深夜休憩や時差出勤時など）は、自動計算されないため「休憩時間修正申請」が必要です,,。
*   **時間有休取得時:** 休憩時間は労働時間にも有休時間にも含まれません。所定休憩時間を削って退社時間を早めることは禁止されています,。

### C. 特殊な勤務・休暇
*   **時間単位有休:**
    *   中抜けの場合：「時間有休申請」＋「私用外出/戻り打刻」が必要です。
    *   遅出・早退利用の場合：休憩時間を除いて計算されます（例：9:00-12:30勤務の場合、12:30以降を全額有休にはできず、休憩1時間を挟んだ13:30-17:30の4時間が有休扱い等は不可）。
*   **遅刻・遅延:**
    *   電車遅延の場合でも、システム上の打刻は「実際に業務を開始した時刻」を入力します。別途「電車遅延申請」と証明書の提出が必要です。

## 4. データ修正フロー（履歴管理）
エクスポートデータに修正履歴が含まれる場合の読み解き方です。

*   **承認前の修正:** 申請を「取り下げ（取り止め）」て、再申請を行います（データは上書き・消去される挙動）,。
*   **承認後の修正:**
    *   「修正申請」または「取消申請」を行います,。
    *   元データには「修正申請元」「取消申請元」というフラグが立ち、新しい申請が承認されると元のデータは「破棄」ステータスになります,。
*   **締め後の修正:**
    *   月次確定後に修正が発生した場合、通常フローでは修正できません。管理者がロックを解除し、修正後に再度承認フローを回す必要があります（給与計算への影響あり）,。

### ※リーダー（勤怠承認者）向け注記（36協定・コンプライアンス）
分析システム側でアラートを出す場合の閾値設定の参考情報です。
*   **残業上限（36協定）:** 月45時間・年360時間が原則的な上限です。
*   **特別条項:** 臨時的な事情がある場合でも、月100時間未満（休日労働含む）、2～6ヶ月平均80時間以内とする必要があります。
*   **健康管理:** 月80時間超の残業は脳・心臓疾患のリスクが高まるとされています。
