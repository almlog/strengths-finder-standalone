提供されたマニュアルおよび資料に基づき、楽楽勤怠で申請可能な届出（申請書）の種類を整理しました。

WEBシステムのマニュアルとして掲載しやすいよう、**「勤怠修正・打刻」「残業・勤務変更」「休暇・休業」「その他」**のカテゴリに分類して記述します。
※企業ごとの設定により名称や利用可否が異なる場合がありますが、資料にあるものを網羅しています。

### 1. 勤怠修正・打刻に関する申請
日々の打刻漏れや時間の修正、遅刻などの事由を届け出るための申請です。

| 申請書名 | 用途・説明 |
| :--- | :--- |
| **打刻忘れ／打刻訂正** | 打刻を忘れた場合や、間違った時間を修正する場合に使用します。 |
| **遅刻・早退申請** | 遅刻や早退をした際に提出します（控除対象となります）。 |
| **電車遅延申請** | 公共交通機関の遅延により遅刻した場合に使用します。「遅延証明書」の提出が必要な場合があります。 |
| **休憩時間修正申請** | 休日出勤や時差出勤などで所定の休憩時間が変更になる場合や、深夜（22:00～翌5:00）に休憩を取得した場合（「深夜休憩申請」）に使用します。 |
| **休憩時間追加申請** | 休憩時間を追加する場合に使用します（基本設定とは別枠）。 |

### 2. 残業・勤務形態の変更に関する申請
定時以外の勤務や、勤務場所・時間の変更を行うための申請です。

| 申請書名 | 用途・説明 |
| :--- | :--- |
| **残業申請** | 定時を超えて業務を行う場合（内勤者など）に提出します。 |
| **早出申請** | 始業時刻より早く業務を開始する場合に提出します。 |
| **早出中抜け時間帯申請** | 早出勤務の時間帯に、休憩などの中抜けが発生する場合に使用します。 |
| **休日出勤申請** | 休日に出勤する場合に提出します。 |
| **時差出勤申請** | 業務都合等で、所定の始業・終業時刻を変更する場合（シフト変更）に使用します。 |
| **直行直帰申請** | 会社に立ち寄らず、外出先へ直接出勤・退勤する場合に使用します。 |
| **出張申請** | 出張を行う場合に使用します。 |
| **在宅勤務申請** | テレワーク（在宅勤務）を行う場合に使用します。 |
| **待機申請** | 携帯待機などを実施した場合に使用します。 |
| **宿直申請** | 宿直勤務を実施した場合に使用します。 |

### 3. 休暇・振替に関する申請
有給休暇や特別休暇、休日の振替を行うための申請です。

| 申請書名 | 用途・説明 |
| :--- | :--- |
| **有休申請** | 年次有給休暇（全日・半日）を取得する場合に使用します。 |
| **時間有休申請** | 年次有給休暇を「時間単位」で取得する場合に使用します。 |
| **欠勤申請** | 有給休暇以外の理由で欠勤する場合に使用します（控除対象）。 |
| **特休申請** | 慶弔休暇など、会社規定の特別休暇を取得する場合に使用します。 |
| **振替出勤申請** | 休日を労働日に変更する場合に、「振替休日申請」とセットで提出します。 |
| **振替休日申請** | 労働日を休日に変更する場合に、「振替出勤申請」とセットで提出します。 |
| **代休申請** | 休日出勤をした代わりに、別の労働日を休みとする場合に使用します。 |
| **生理休暇申請** | 生理休暇を取得する場合に使用します。 |
| **積立休暇申請** | 失効した有給休暇を積み立てたものを使用する場合の申請です。 |
| **明け休申請** | 夜勤明けなどで勤務免除（明け休）となる場合に使用します。 |

### 4. 看護・介護に関する申請
法律に基づく子の看護や家族の介護のための休暇申請です。

| 申請書名 | 用途・説明 |
| :--- | :--- |
| **子の看護休暇申請** | 子の看護のために休暇を取得する場合に使用します。 |
| **時間子の看護休暇申請** | 子の看護休暇を「時間単位」で取得する場合に使用します。 |
| **介護休暇申請** | 家族の介護のために休暇を取得する場合に使用します。 |
| **時間介護休暇申請** | 介護休暇を「時間単位」で取得する場合に使用します。 |

### 5. 月次締め処理
| 申請書名 | 用途・説明 |
| :--- | :--- |
| **出勤簿提出** | 1ヶ月の勤務実績が確定した後、承認者に提出する最終的な申請です（締め処理）。 |
